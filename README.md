# Terminal Trade Bot 🚀

Экономическая игра-трейдер в Telegram

## 📋 Команды

### Для всех:
- `/start` - начать игру
- `/help` - помощь
- `/tbuy AAPL 5` - купить 5 акций Apple
- `/tsell AAPL 5` - продать 5 акций Apple
- `/tnext` - следующий день
- `/tstats` - твоя статистика
- `/tlist` - список компаний
- `/tleader` - таблица лидеров
- `/tpromo КОД` - активировать промокод
- `/shop` - магазин
- `/inventory` - инвентарь
- `/cubes` - твои кубки
- `/achievements` - достижения

### Для админов:
- `/admin give money ID 5000` - выдать деньги
- `/admin take money ID 2000` - забрать деньги
- `/admin give cubes ID 3` - выдать кубки
- `/admin take cubes ID 1` - забрать кубки
- `/admin give item ID booster` - выдать предмет
- `/admin reset ID` - сброс пользователя
- `/admin info ID` - информация
- `/admin stats` - статистика
- `/admin mail "текст"` - рассылка
- `/admin promo create CODE 500 10` - создать промокод
- `/admin promo delete CODE` - удалить промокод
- `/admin season create "Лето" 30` - создать сезон
- `/admin season end` - завершить сезон

## 🛠 Установка на BotHost

1. Загрузи все файлы на GitHub
2. На BotHost укажи репозиторий
3. Команда запуска: `python bot.py`
4. Готово!
